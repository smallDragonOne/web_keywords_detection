# Page Watcher Summary (MV3)

一个可本地开发模式加载的 Chrome/Edge 扩展，用于在已打开标签页中选择多个页面进行监听，自动刷新抓取新内容，按关键词过滤并持久化展示摘要列表。

## 功能

- 在任意页面可直接点击「加入监听当前标签页」，再次点击可移除
- 默认不监听，点击「开始监听」后才工作
- 按设置的刷新间隔自动刷新监听中的标签页后抓取内容
- 支持关键词过滤（分号分隔，例如 `融资;发布;update`）
- 去重抓取（相同内容不会重复入库）
- 面板提醒（未读命中数量与最近提醒时间）
- 支持导出 JSON / CSV（CSV 可直接用 Excel 打开）
- 面板显示「监听中的标签页」状态，标签页关闭会自动移除
- 命中后保存摘要信息：
  - 页面 URL
  - 是否疑似包含详情页链接
  - 页面内时间（time 标签或正文时间格式）
  - 简短内容片段
- 命中列表持久化保存到本地 `chrome.storage.local`

## 本地加载

1. 打开浏览器扩展管理页面
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. 打开「开发者模式」
3. 选择「加载已解压的扩展程序」
4. 选择当前目录 `browerEx`

## 使用说明

1. 点击扩展图标打开面板
2. 配置：
   - 在目标页面打开插件，点击「加入监听当前标签页」
   - 关键词：例如 `融资;发布;update`
   - 自动刷新间隔：最小 1 分钟
   - 最大保留条数：用于控制本地列表大小
   - 是否开启面板提醒
3. 点击「保存设置」
4. 点击「开始监听」开始工作（默认未监听）
5. 在「监听中的标签页」查看当前监听状态
6. 在「命中摘要」查看筛选结果
7. 可导出 JSON 或 Excel(CSV)
8. 可点击「提醒清零」和「清空列表」

## 目录

- `manifest.json` - 扩展清单（MV3）
- `background.js` - 后台扫描与匹配逻辑
- `popup.html` - 面板结构
- `popup.css` - 面板样式
- `popup.js` - 面板交互与列表渲染
